# seu_traje
## Prepare environment
### Development
```sh
$ cd seu_traje  
$ npm run-script installDevelopment
```
### Staging
```sh
$ cd seu_traje  
$ npm run-script installStaging
```
## Run
### Development
```sh
$ cd seu_traje
$ npm run-script startDevelopment
```
### Staging
```sh
$ cd seu_traje  
$ npm run-script startStaging
```


To commit without running the GitLab CI add `[ci skip]` to your commit message.
